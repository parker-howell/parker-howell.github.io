3.1 Real Numbers

Def The function \(f: X \rightarrow Y\) is invertible if for each \(y \in Y\) there exists a unique \(x \in X\) such that \(f(x) = y\).

Def Fix \(X \subseteq \mathbb{R}\) nonempty.

  1. Call \(u \in \mathbb{R}\) an upper bound of \(X\) if, for each \(x \in X, u \geq x\)
  2. Call \(l \in \mathbb{R}\) a lower bound of \(X\) if, for each \(x \in X, x \geq l\)

Def Call \(u^*\) a least upper bound (supremum) of \(X\) if \(u^*\) is an upper bound of \(X\) and, for each upper bound \(u, u \geq u^*\). Denoted \(\sup X\).

Remark \(\sup X\) and \(\inf X\) need not be members of \(X\). Proof: Consider \([0,1) = X\). Then \(sup X = 1\). And \(1 \geq X\) for \(x \in X\). \(1 > u\) there exists \(1 > x > u\) and so \(u\) is not an upper bound.

Completeness Axiom: Let \(X \subseteq \mathbb{R}\) be nonempty. If \(X\) is bounded above (resp bounded below) then \(X\) has a least upper bound (resp a greatest lower bound.)

Notational convention: If \(X\) nonempty and has an upper bound, then \(\sup X \in \mathbb{R}\) exists. But if \(X\) nonempty and does not have an upper bound, we write \(\sup X = \infty\).

Proposition 1: Suppose \(X \subseteq \mathbb{R}\) is nonempty and has an upper bound. Then

  1. \(\sup X\) is unique
  2. for each \(\varepsilon >0\) there exists \(x \in X\) such that \(\sup X - \varepsilon < x\).

Proof

  1. fix \(u_1, u_2\) that are both least upper bounds of \(X\). Then \(u_1 \leq u_2\), since \(u_1\) is a least upper bound. Similarly, \(u_2 \leq u_1\), since \(u_2\) is a least upper bound. Hence, \(u_1=u_2\).
  2. By contradiction. Suppose not. Then there exists \(\varepsilon > 0\) such that for all \(x \in X, \sup X - \varepsilon \geq x\). Then \(\sup X - \varepsilon\) is an upper bound with \(\sup X - \varepsilon < \sup X\) contradicting \(\sup X\) being supremum.

Def Fix \(X \subseteq \mathbb{R}\) nonempty

  1. Call \(\bar{x}\) a max of \(X\) if \(\bar{x} \in X\) and \(\bar{x} \geq x\) for each \(x \in X\)
  2. Call \(\underline{x}\) a min of \(X\) if \(\underline{x} \in X\) and \(\underline{x} \leq x\) for each \(x \in X\)

Example: [0,1) doesn’t have a max, even though it’s bounded

Proposition 2 Fix \(X \subseteq \mathbb{R}\) nonempty

  1. \(\bar{x}\) is a max of \(X\) iff \(\bar{x}=\sup X \in X\)
  2. \(\underline{x}\) is a max of \(X\) iff \(\underline{x}=\inf X \in X\)

Pf: a) \(\bar{x} \in X\), we know this from the def of the max. b) need to show \(\bar{x}=\sup X\). Well, we know \(\bar{x}\) is an upper bound and there exists a least upper bound \(=sup X\) by the completeness axiom. Since \(\bar{x} \in X\) each upper bound of \(X\) is greater than or equal to \(\bar{x}\). Hence, \(\bar{x}=\sup X\).

Corollary 1: Fix \(X \subseteq \mathbb{R}\) nonempty. Then \(X\) has a max iff \(\sup X \in X\). Moreover, if \(X\) has a max, it is unique.

A real-valued function, \(f: X \rightarrow \mathbb{R}\). Call \(f\) bounded if \(f(X) \subseteq \mathbb{R}\) is bounded.

3.2 Metric Spaces

Def A metric on \(X\) is a function \(d: X \times X \rightarrow \mathbb{R}\) such that

  1. for each \(x,y\in X, d(x,y) \geq 0\)
  2. for each \(x,y\in X, d(x,y) = 0\) iff \(x=y\)
  3. for each \(x,y\in X, d(x,y)=d(y,x)\) (symmetry)
  4. for each \(x,y,z\in X, d(x,z) \leq d(x,y) + d(y,z)\) (triangle inequality)

Def A metric space is a pair \((X,d)\) where \(X\) is a nonempty set and \(d\) is a metric on \(X\).

Example 1: Let \(X \subseteq \mathbb{R}\) nonempty. The usual metric is \(d^u: X \times X \rightarrow \mathbb{R}\) such that \(d^u(x,y) = |x-y|\).

Example 2: \(X \subseteq \mathbb{R}^n\) nonempty. The euclidean metric \(d^{euc}: X \times X \rightarrow \mathbb{R}\) is \(d^{euc}(\vec{x}, \vec{y}) = \sqrt{\Sigma^n_{i=1} (x_i-y_i)^2}\)

Note that when \(n=1\) we have \(d^u(x,y)=d^{euc}(x,y)\)

Any time we look at a subset of \(\mathbb{R}^n\) we’re going to impose euclidean distance, unless we are explicitly told otherwise.

Example 3: “taxicab” metric. \(X \subseteq \mathbb{R}^n\) nonempty: \(d^t(\vec{x}, \vec{y}) = \sum_{i=1}^n |x_i - y_i|\)

\(d^t(\vec{x}, \vec{y}) \geq d^{euc}(\vec{x}, \vec{y})\).

Example 4: Fix some nonempty set, \(X\). Discrete metric: \(d^{disc} (x,y) = \{1 \text{ if } x \neq y, 0 \text{ if } x=y\)

Example 5: \((X, \rho), (\mathbb{R}, d^u)\), where \(\rho: X\times X \rightarrow \mathbb{R}\) is a metric.

\(F^B(X, \mathbb{R}) =\) set of bounded \(f: X \rightarrow \mathbb{R}\)

\(d: F^B(X, \mathbb{R}) \times F^B(X, \mathbb{R}) \rightarrow \mathbb{R}\) such that \(d(f,g) = \sup_{x\in X} d^u(f(x), g(x)) = \sup_{x\in X} |f(x) - g(x)|\)

3.3 Sequences

Def Fix a metric space \((X,d)\). A sequence on \((X,d)\) is a function \(f: \mathbb{N} \rightarrow X\).

Def Fix a sequence \((x_n:n\in \mathbb{N})\) on \((X,d)\). The limit of \((x_n:n\in \mathbb{N})\) is some \(x \in X\) such that for each \(\varepsilon >0\) there exists \(N\) with \(d(x_n, x)< \varepsilon\) for all \(n\geq N\).

The sequence converges if it has a limit in \(X\). The sequence diverges if it does not have a limit in \(X\).

Proposition 1: Any sequence has at most one limit.

Proof: Fix a sequence \((x_n: n \in \mathbb{N})\) and suppose that \(x, y\) are both limits of the sequence.

want to show: x = y

We know

  • there exists \(N(x)\) such that for all \(n \geq N(x), d(x_n, x) < \varepsilon / 2\)
  • there exists \(N(y)\) such that for all \(n \geq N(y), d(x_n, y) < \varepsilon / 2\)

Let \(N = \max\{N(x), N(y)\}\).

For each \(n \geq N\), \(d(x,y) \geq d(x, x_n) + d(x_n,y) < \varepsilon /2 + \varepsilon / 2 = \varepsilon\).

So, \(d(x,y) < \varepsilon\) for all \(\varepsilon > 0\). So, \(d(x,y)=0\). So, \(x=y\). QED.

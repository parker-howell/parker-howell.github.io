11 Matrices as transformations

11.1 Linear Transformations

11.1.1 Def “linear transformation”

A linear transformation from \(\mathbb{R}^m\) to \(\mathbb{R}^n\) is \(L: \mathbb{R}^m \rightarrow \mathbb{R}^n\) such that

  1. for each \(\vec{x}, \vec{y} \in \mathbb{R}^m, L(\vec{x} + \vec{y}) = L(\vec{x}) + L(\vec{y})\)
  2. for each \(\lambda \in \mathbb{R}\) and each \(\vec{x} \in \mathbb{R}^m, L(\lambda\vec{x}) = \lambda L(\vec{x})\)

Example 1 \(L: \mathbb{R} \rightarrow \mathbb{R}\) is a linear transformation. For each \(x \in \mathbb{R}, L(x) = L(1x) = xL(1)\)

\[\Rightarrow x\in \mathbb{R}, L(x) = cx.\]

Other examples omitted. See paper notes.

11.1.2 Prop 1 “linear transformation properties”

Let \(L: \mathbb{R}^m \rightarrow \mathbb{R}^n\) be a linear transformation. Then

  1. \(L(\vec{0}) = \vec{0}\)
  2. If \(X = \{\vec{x}_1, ..., \vec{x}_k\} \subseteq \mathbb{R}^m\) is linearly dependent, then \(L(X)\) is linearly dependent.
  3. \(L\) is injective if and only if \(L(\vec{x}) = \vec{0} \Rightarrow \vec{x} = \vec{0}\)
  4. \(L\) is surjective if and only if \(span(\{L(\vec{e}_1), ..., L(\vec{e}_m)\}) = \mathbb{R}^n\)

11.1.2.1 Term “matrix representation” of \(L\)

For each \(L: \mathbb{R} \rightarrow \mathbb{R}\), linear transformation, there exists a unique \(A \in \mathbb{R}^{n \times m}\) such that \[L(\vec{x}) = A \vec{x}, \forall \vec{x} \in \mathbb{R}^m.\] We call \(A\) the matrix representation of \(L\).

11.1.3 Prop 2 “invertible iff bijective”

Suppose \(A \in \mathbb{R}^{m \times n}\) is the matrix representation of \(L: \mathbb{R}^m \rightarrow \mathbb{R}^n\). The \(A\) is invertible if and only if \(L\) is bijective.

11.1.4 Def “eigenvector”

An eigenvector of \(A \in \mathbb{R}^{n \times n}\) is a vector \(\vec{x} \in \mathbb{R}^n\) with \(\vec{x} \neq \vec{0}\) such that \(A\vec{x} = \lambda \vec{x}\) for some \(\lambda \in \mathbb{R}\).

\(\lambda\) is the eigenvalue for \(\vec{x}\).

Example 8 \[ A = \begin{bmatrix} 0 \\ . \\ . \\ . \\ a_{ii} . \\ . \\ . \\ 0 \end{bmatrix} = a_{ii}\vec{e}_i \] NOT FINISHED

Example 9

\[A = \begin{bmatrix} 0 & 1 \\ 1 & 1 \end{bmatrix}\]

\[A - \lambda Id^2 = \begin{bmatrix} 0 & 1 \\ 1 & 1 \end{bmatrix} + \begin{bmatrix} -\lambda & 0 \\ 0 & -\lambda \end{bmatrix} = \begin{bmatrix} -\lambda & 1 \\ 1 & 1 - \lambda \end{bmatrix}\]

Next, \[\det(\begin{bmatrix} -\lambda & 1 \\ 1 & 1 - \lambda \end{bmatrix}) = -\lambda(1-\lambda) - 1 = -\lambda + \lambda^2 - 1 \Rightarrow \lambda = \frac{1 \pm \sqrt{5}}{2}\]

Solving this system of equations eventually shows \(x_1 = \frac{\sqrt{5} - 1}{2}, x_2 = 1\)

11.1.4.1 Notice: If 0 is an eigenvalue:

\[0 = det(A - \lambda Id^n) = det(A)\]

11.1.5 Prop 4 “invertible iff eigenvalue nonzero”

A matrix \(A \in \mathbb{R}^{n \times n}\) is invertible if and only if 0 is not one of its eigenvalues.

11.2 Quadratic Forms

11.2.1 Def “quadratic forms”

A quadratic form is a mapping \(q: \mathbb{R}^n \rightarrow \mathbb{R}\) such that \[q(\vec{x}) = \sum_{i=1}^n\sum_{j=1}^n a_{ij}x_i x_j\]

  • if n=1: \(q(\vec{x} = x_1) = a_{11}x_1^2\)
  • if n=2: \(q(\vec{x}) = a_{11}x_1^2 + a_{12}x_1 x_2 + a_{21}x_1 x_2 + a_{12}x_2^2\)

properties when \(n=1\):

  1. if \(a_{11} > 0\) then \(q(x_1)>0 \forall x_1 \neq 0\) and \(q(x_1) \geq 0 \forall x\)
  2. if \(a_{11} < 0\) then \(q(x_1)<0 \forall x_1 \neq 0\) and \(q(x_1) \leq 0 \forall x\)

11.2.2 Def “quadratic positive/negative”

the quadratic form \(q: \mathbb{R}^n \rightarrow \mathbb{R}\) is

  1. positive (resp negative) definite if \(q(\vec{x})>0\) for all \(\vec{x} \neq \vec{0}\)
  2. positive (resp negative) semi-definite if \(q(\vec{x}) \geq 0\) for all \(\vec{x}\)
  3. indefinite if there exists \(\vec{x}, \vec{y}\) s.t. \(q(\vec{x}) < 0\) and \(q(\vec{y}) > 0\)

11.2.3 Prop 5 “unique symmetric matrix”

Let \(q: \mathbb{R}^n \rightarrow \mathbb{R}\) be a quadratic form. There exists a unique symmetric matrix \(A \in \mathbb{R}^{n \times n}\) such that for each \(\vec{x} \in \mathbb{R}^n\) \[q(\vec{x}) = \vec{x}^T A \vec{x}.\]

11.2.4 Prop 6 “eigenvalues pos \(\iff\) pos def”

Let \(q(\vec{x}) = \vec{x}^T A \vec{x}\) be a quadratic form. \(A\) MUST ALSO BE SYMMETRIC!

  1. \(q\) (or \(A\)) is positive (resp. negative) definite if and only if all of its eigenvalues are strictly positive (negative)
  2. \(q\) (or \(A\)) is positive (resp. negative) semidefinite if and only if all of its eigenvalues are \(\geq 0\) (\(\leq 0\)).

11.2.4.1 Corollary

Fix \(A \in \mathbb{R}^{n \times n}\).

  1. if \(A\) is positive (negative) definite, it is invertible.
  2. if \(A\) is positive (negative) definite, \(A^{-1}\) is also positive (negative) definite
  3. if \(A\) is positive (negative) semidefinite, but not definite, then \(A\) is NOT invertible.

11.2.5 Prop 7 “pos/neg \(\iff\) leading principal minor”

Fix \(A \in \mathbb{R}^{n \times n}\). \(A\) MUST ALSO BE SYMMETRIC!

  1. \(A\) is positive definite if and only if, for each \(r= 1, ..., n\), the leading principal minor of order \(r\) is strictly positive.
  2. \(A\) is positive semi-definite if and only if, for each \(r= 1, ..., n\), each principal minor of order \(r\) is greater than or equal to zero.
  3. \(A\) is negative definite if and only if, for each \(r= 1, ..., n\), the leading principal minor of order \(r\) has sign \((-1)^r\).
  4. \(A\) is negative semi-definite if and only if, for each \(r= 1, ..., n\), each principal minor of order \(r\) either has sign \((-1)^n\) or is zero.

11.2.5.1 Term “lower triangle matrix”

\(A\) is a lower triangle matrix if \(a_{ij} = 0\) when \(i<j\)

11.2.6 Thm “Cholesky Decomposition”

Let \(A \in \mathbb{R}^{n \times n}\) symmetric matrix. If \(A\) is positive definite, there exists a unique invertible lower triangle matrix \(B \in \mathbb{R}^{n \times n}\) such that \(A = BB^T\).

We call \(B\) the Cholesky Decomposition of \(A\).

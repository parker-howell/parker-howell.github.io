5 Sequences

5.0.1 Def “sequence”

Fix a metric space \((X,d)\). A sequence on \((X,d)\) is a function \(f: \mathbb{N} \rightarrow X\).

5.0.2 Def “limit”

Fix a sequence \((x_n:n\in \mathbb{N})\) on \((X,d)\). The limit of \((x_n:n\in \mathbb{N})\) is some \(x \in X\) such that for each \(\varepsilon >0\) there exists \(N\) with \(d(x_n, x)< \varepsilon\) for all \(n\geq N\).

The sequence converges if it has a limit in \(X\). The sequence diverges if it does not have a limit in \(X\).

5.0.3 Prop 1 “sequences have max one limit”

Any sequence has at most one limit.

Proof: Fix a sequence \((x_n: n \in \mathbb{N})\) and suppose that \(x, y\) are both limits of the sequence.

want to show: x = y

We know

  • there exists \(N(x)\) such that for all \(n \geq N(x), d(x_n, x) < \varepsilon / 2\)
  • there exists \(N(y)\) such that for all \(n \geq N(y), d(x_n, y) < \varepsilon / 2\)

Let \(N = \max\{N(x), N(y)\}\).

For each \(n \geq N\), \(d(x,y) \geq d(x, x_n) + d(x_n,y) < \varepsilon /2 + \varepsilon / 2 = \varepsilon\).

So, \(d(x,y) < \varepsilon\) for all \(\varepsilon > 0\). So, \(d(x,y)=0\). So, \(x=y\). QED.

5.0.4 Def “Limit”

A limit of \((x_n, n\in \mathbb{N})\) is some \(x \in X\) such that for each \(\varepsilon >0\), there exists \(N\) with \(d(x_n, x) < \varepsilon\) for all \(n \geq N\).

The sequence converges if it has a limit in \(X\). Otherwise it diverges.

Special case of interest: real-valued sequence (whenever \(X\) is a subset of the reals), \(X \subseteq \mathbb{R}, d = d^u\)

A sequence is just a mapping of the natural numbers to \(X\). (i.e., \(f: \mathbb{N} \rightarrow X\))

Example 1: \((1/n, n \geq 1)\) on \((\mathbb{R}, d^u)\).

\(\lim_{n \rightarrow \infty} 1/n = 0\): Fix \(\varepsilon >0\). To show: there exists \(N\) such that \[|1/n - 0| < \varepsilon\] for all \(n \geq N\). Take \(N = 1/\varepsilon\). Rearrange to get \(1/N < \varepsilon\). And clearly, any \(n > N\) will also satisfy this inequality. So, we have proved that the limit converges to 0.

Example 1b \((1/n, n\geq 2)\) on \(((0,1), d^u)\)

This diverges to 0. It gets closer to 0, but the limit is not in our set, so it’s divergent.

Example 2 \((n: n\geq 1)\) on \((\mathbb{R}, d^u)\) Can there exist some \(x \in \mathbb{R}\) such that for each \(\varepsilon > 0\) there exists \(N\) where \(|n-x| < \varepsilon\) for all \(n \geq N\)?

No. For each \(\varepsilon\) there is some \(N\) with \(N > x + \varepsilon\). I also know for every \(n \geq N, |n - x| > \varepsilon\).

5.0.5 Def “divergence”

Fix a real-valued sequence, \((x_n: n\geq 1)\) on \((X, d^u)\).

  1. \((x_n: n\geq 1)\) diverges to \(\infty\) if, for each \(k\in \mathbb{R}\), there exists \(N\) such that \(x_n>k\) for all \(n\geq N\).
  2. \((x_n: n\geq 1)\) diverges to \(-\infty\) if, for each \(k\in \mathbb{R}\), there exists \(N\) such that \(x_n<k\) for all \(n\geq N\).

It’s convenient to write \(lim_{n\rightarrow \infty} x_n = \infty\). But this is an abuse of notation. Just remember that it doesn’t actually converge (bc you can’t converge outside your set).

Case \((\mathbb{R}^m, d^{euc})\), \((x_n: n \in \mathbb{N})\) where \(x_n \in \mathbb{R}^m\). So \(x_n = (x_{n1}, x_{n2}, ..., x_{nm}) \in \mathbb{R}^m\)

Result the sequence \((x_n: n\in\mathbb{N})\) converges to \(x^*\in\mathbb{R}^m\) iff for each \(i=1, ..., m\), \((x_{ni}: n\in \mathbb{N})\) converges to \(x_i^*\) in \((\mathbb{R}, d^u)\)

Case \((X, d^{disc})\)

the seq \((x_n: n \in \mathbb{N})\) converges to \(x\in X\) iff there exists some \(n\geq N\) such that \(x_n=x\) for all \(n\geq N\).

Suppose \((x_n: n \in \mathbb{N})\) converges to \(x\in X\). Then, there exists some \(N\) such that \(1/2 > d^{disc}(x_n,x) \Rightarrow x_n=x\) for all \(n \geq N\) since \(d^{disc}(x_n,x) \in \{0,1\}\) (note that 1/2 is arbitrary–any value between 0 and 1 would have worked here.)

5.1 Subsequences

seq \(= (1,6, 2, 9, 7, 5, 22, 1, ...)\)

subsequence \(= (1,6, 9, 5, 22, ...)\)

5.1.1 Def “subsequence”

Fix a sequence \((x_n: n\in \mathbb{N})\). Call \((y_n: n\in \mathbb{N})\) a subsequence if there exists a function \(f: \mathbb{N}\rightarrow \mathbb{N}\) such that for each \(k\in \mathbb{N}\),

  1. \(f(k+1) > f(k)\geq k\),
  2. \(y_k=x_{f(k)}\)

5.1.2 Prop 2 “sequence converges \(\Rightarrow\) subsequences converge”

Let \((x_n: n\in \mathbb{N})\) be a sequence on \((X,d)\) that converges to \(x\). Then each subsequence of \((x_n: n\in \mathbb{N})\) also converges to \(x\).

Proof Let \((y_n: n\in \mathbb{N})\) be a subsequence of \((x_n: n\in\mathbb{N})\) associated with a function \(f: \mathbb{N}\rightarrow \mathbb{N}\). Since \((x_n: n\in\mathbb{N})\) converges to \(x\) for each \(\varepsilon>0\) there exists \(N\) such that \(d(x_n, x) < \varepsilon\), for all \(n \geq N\). There exists \(K\) such that \(f(K) \geq N\). Then it follows \(d(y_k, x) = d(x_{f(k)}, x) < \varepsilon\) for all \(k \geq K\) since \(f(k) \geq N\).

Example 3 \(((-1)^n: n\geq 1)\). Pick two subsequences:

  1. \((-1, -1,..., ) \rightarrow\) converges to -1
  2. \((1, 1,..., ) \rightarrow\) converges to 1

but a sequence cannot have two limits, so this sequence must diverge.

5.1.3 Def “bounded sequence”

Call a sequence \((x_n: n\in \mathbb{N})\) bounded if there exists \(y\in X\) and \(M>0\) such that \(d(x_n, y) \leq M\) for all \(n\in \mathbb{N}\).

5.1.4 Prop 3 “A convergent sequence is bounded.”

Proof Omitted

Caution: not every bounded sequence is convergent (example: \((-1)^n\) on \(X =\mathbb{R}, d^u\))

5.1.5 Def “increasing”

Let \((x_n: n\in \mathbb{N})\) be a real-valued sequence. The sequence is increasing if for each \(n \in \mathbb{N}, x_{n+1} \geq x_n\).

5.1.6 Def “monotone”

The sequence is monotone if it is either increasing or decreasing.

5.1.7 Prop 4 “bounded, monotone \(\Rightarrow\) convergent”

If \((x_n: n\in \mathbb{N})\) is a bounded, monotone sequence on \((\mathbb{R}, d^u)\) then it is convergent.

Proof Fix \((x_n: n\in \mathbb{N})\) on \((\mathbb{R}, d^u)\) that is bounded, monotone sequence. Without loss of generality, assume it is increasing.

Since bounded, the set \(\{x_n: n\in\mathbb{N}\}\) has to be bounded above (note: switching to set so we can use completeness axiom. apparently that may not apply to sequences, but it does to sets). By completeness axiom, if there is an upper bound, there must be a least upper bound, \(x\in \mathbb{R}\).

Want to show: \(x=\lim x_n\)

To show: Fix \(\varepsilon >0\), Find some \(N\) such that \(|x_n-x| < \varepsilon\), for all \(n \geq N\).

Note: there exists \(N\) such that \(x_N > x-\varepsilon\), by Prop 1 of the real numbers notes.

Since the seq is increasing \(x_n \geq x_N \geq x-\varepsilon\). Rearrange to get \(\varepsilon > x-x_n\). Since \(x\) is an upper bound (the supremum) we have \(x > x_n\) which means \(x-x_n = |x-x_n|\).

So, we have \(\varepsilon > |x-x_n|\). QED.

5.1.8 Prop 5 “squeeze thm”

Let \((x_n: n\in \mathbb{N}), (y_n: n\in \mathbb{N}), (z_n: n\in \mathbb{N})\) be sequences on \((\mathbb{R}, d^u)\). If, for each \(n\), \(x_n \leq y_n \leq z_n\) and \(\lim_{n\rightarrow\infty} x_n = \lim_{n\rightarrow\infty} z_n = a\) then \(\lim_{n\rightarrow\infty} y_n = a\)

Note: even if \(x_n\) and \(z_n\) crossed over each other at some points, I might still be able to use the squeeze thm by using subsequences to satisfy the fact that for each \(n\), \(x_n \leq y_n \leq z_n\)

5.1.9 Prop 6 “limit algebra”

Let \((x_n: n\in \mathbb{N}), (y_n: n\in \mathbb{N})\) be sequences on \((\mathbb{R}, d^u)\) with \(\lim_{n\rightarrow\infty} x_n = x, \lim_{n\rightarrow\infty} y_n = y\). Then

  1. \(\lim_{n\rightarrow\infty} |x_n| = |x|\)
  2. \(\lim_{n\rightarrow\infty} x_n + y_n = x+y\)
  3. \(\lim_{n\rightarrow\infty} x_ny_n = xy\)
  4. If \(x_n \neq 0 \forall n\) and \(x\neq 0\), then \(\lim_{n\rightarrow\infty} y_n/x_n= y/x\)

5.1.10 Prop 7: Bolzano-Weierstrauss Thm “bounded seq \(\Rightarrow\) convergent subsequence”

Every bounded sequence on \((\mathbb{R}, d^u)\) has a convergent subsequence.

5.1.11 Def “Cauchy”

Call a sequence Cauchy if for each \(\varepsilon >0\) there exists \(N\) such that for all \(n, m \geq N, d(x_n, x_m) < \varepsilon\).

5.1.12 Prop 8 “convergent \(\Rightarrow\) Cauchy”

Any convergent sequence is Cauchy.

Proof Fix a sequence \((x_n: n\in \mathbb{N})\) with \(\lim_{n\rightarrow\infty}x_n=x\). So there exists \(g: (0,\infty) \rightarrow \mathbb{N}\) such that for each \(\varepsilon >0\) and \(n\geq g(\varepsilon), d(x_n,x) < \varepsilon\). To show it’s Cauchy, fix \(\varepsilon >0\) and choose \(N = g(\varepsilon/2)\). By triangle inequality I know \(d(x_n,x_m) \leq d(x_n, x) + d(x, x_m) < \varepsilon/2 + \varepsilon/2 = \varepsilon\).

Remark: a seq can be Cauchy without converging. Consider example 1b: \((1/n, n\geq 2)\) on \(((0,1), d^u)\). This doesn’t converge (because 1 isn’t in the metric space), but it is cauchy. (i.e., for \(\varepsilon >0\) there exists \(N\) such that \(|1/n -1/m| < \varepsilon\) for all \(n, m \geq N\).)

5.1.13 Def complete space

A metric space is complete if every Cauchy sequence converges.

Examples of complete metric spaces:

  • \((\mathbb{R}^n, d^{euc})\)
  • \((X, d^{disc})\) for \(X\) finite
  • \(((0,1), d^u)\) is NOT complete
  • \((\mathbb{Q}, d^u)\) is NOT complete

Remark: All Cauchy sequences converge in complete spaces.

5.1.14 Prop 9 “Cauchy \(\Rightarrow\) bounded”

Every Cauchy sequence is bounded.

8.0.1 Def “pointwise convergence”

Let \((f_n: n \geq 1)\) be a sequence of functions in \(F(X, \mathbb{R})\). Say \(f_n: n\geq 1\) converges pointwise if there exists \(f \in F(X,\mathbb{R})\) such that for each \(x \in X, \lim_{n\rightarrow\infty} f_n(x) = f(x)\).

So, in this case, say \((f_n: n\geq 1)\) converges pointwise to \(f\).

Example 2 Let \(f_n: [k-1,k] \rightarrow \mathbb{R}\) such that \(f_n(x) = \frac{1}{nx}\). For each \(x \in [k-1, k]\) we have that \(\lim_{n\rightarrow \infty} f_n(x)=0.\) So \(f_n\) converges pointwise to \(f\) where \(f(x)=0\) for all \(x \in [k-1,k]\).

8.0.2 Def “uniform convergence”

Let \((f_n: n\geq 1)\) be a sequence of functions in \(F(X, \mathbb{R})\). Say \((f_n: n\geq 1)\) converges uniformly to \(f \in F(X, \mathbb{R})\) if for each \(\varepsilon > 0\) there exists \(N\) such that for all \(n\geq N\) and each \(x \in X\), \(|f_n(x) - f(x)| < \varepsilon\).

In this case, call \(f\) the uniform limit of \((f_n: n \geq 1)\)

8.0.2.1 Remember difference between uniform and pointwise convergence:

  • uniform convergence: \(N\) cannot depend on \(x\in X\)
  • pointwise convergence: \(N\) can depend on \(x \in X\)

Given a function \(f: X \rightarrow \mathbb{R}\) and some nonempty \(E \subseteq X\), write \(f_{|E}\) for a function \(g: E \rightarrow \mathbb{R}\) with \(g(x)=f(x)\) for all \(x\in E\). Call this “the restriction of \(f\) to \(E\).

8.0.3 Prop 1 “restricting domain doesn’t change uniform convergence”

Let \((f_n: n \geq 1)\) be a sequence of functions in \(F(X, \mathbb{R})\) that converges uniformly to \(f \in F(X, \mathbb{R})\). For each non-empty \(E \subseteq X\), \((f_n|E: n \geq 1)\) is a sequence of functions in \(F(E,\mathbb{R})\) that converges uniformly to \(f_{|E} \in F(E, \mathbb{R})\).

8.0.3.1 Remember: increasing the domain can screw up uniform convergence.

8.0.4 Prop 2 “if fn is cont then f is cont”

Let \((f_n: n \geq 1)\) be a sequence of functions in \(F(X, \mathbb{R})\).

  1. if, for each \(n\), \(f_n\) is continuous at \(x^*\) then \(f\) is continuous at \(x^*\)
  2. if, for each \(n\), \(f_n\) is continuous then \(f\) is continuous

8.0.5 Prop 3 “if seq is uni cont and uni conv then each fn converges to f(x)”

Let \((f_n: n \geq 1)\) be a sequence on \(F(X, \mathbb{R})\) where each \(f_n\) is uniformly continuous and \((f_n: n\geq1)\) converges uniformly to \(f \in F(X, \mathbb{R})\). If \((x_n: n\geq1)\) is a sequence on \(X\) that converges to \(x \in X\), then \((f_n(x_n): n \geq 1)\) converges to \(f(x)\).

Reminder: “Uniform metric”: \(d\) on \(F^B(X, \mathbb{R})\): \[d(f,g) = \sup_{x\in X} |f(x)-g(x)|\]

Convergence under the uniform metric:

Fix a seq \((f_n: n \geq 1)\) on \((F^B(X, \mathbb{R}),d)\). Then \((f_n: n \geq 1)\) converges to \(f\) if for each \(\varepsilon > 0\) there exists some \(N\) such that \(d(f_n, g) = \sup|f_n(x) - f(x)| < \varepsilon\) for all \(n \geq N\)

*Note that \(|f_n(x) - f(x)| < \varepsilon\) is the same here as it was in the definition of uniform convergence.

8.0.6 Prop 4 “converges uniformly \(\iff\) converges under uniform metric”

Fix a seq of functions \((f_n: n \geq 1)\) on \(F^B(x, \mathbb{R})\) the sequence converges uniformly to \(f \in F^B(X, \mathbb{R})\) if and only if it converges to \(f\) under the uniform metric.

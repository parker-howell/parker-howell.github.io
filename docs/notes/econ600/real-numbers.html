3 Real Numbers

3.0.1 Def “bounds”

Fix \(X \subseteq \mathbb{R}\) nonempty.

  1. Call \(u \in \mathbb{R}\) an upper bound of \(X\) if, for each \(x \in X, u \geq x\)
  2. Call \(l \in \mathbb{R}\) a lower bound of \(X\) if, for each \(x \in X, x \geq l\)

3.0.2 Def “supremum”

Call \(u^*\) a least upper bound (supremum) of \(X\) if \(u^*\) is an upper bound of \(X\) and, for each upper bound \(u, u \geq u^*\). Denoted \(\sup X\).

Remark \(\sup X\) and \(\inf X\) need not be members of \(X\). Proof: Consider \([0,1) = X\). Then \(sup X = 1\). And \(1 \geq X\) for \(x \in X\). \(1 > u\) there exists \(1 > x > u\) and so \(u\) is not an upper bound.

3.0.3 Completeness Axiom:

Let \(X \subseteq \mathbb{R}\) be nonempty. If \(X\) is bounded above (resp bounded below) then \(X\) has a least upper bound (resp a greatest lower bound.)

Notational convention: If \(X\) nonempty and has an upper bound, then \(\sup X \in \mathbb{R}\) exists. But if \(X\) nonempty and does not have an upper bound, we write \(\sup X = \infty\).

3.0.4 Prop 1: “sup is unique”

Suppose \(X \subseteq \mathbb{R}\) is nonempty and has an upper bound. Then

  1. \(\sup X\) is unique
  2. for each \(\varepsilon >0\) there exists \(x \in X\) such that \(\sup X - \varepsilon < x\).

Proof

  1. fix \(u_1, u_2\) that are both least upper bounds of \(X\). Then \(u_1 \leq u_2\), since \(u_1\) is a least upper bound. Similarly, \(u_2 \leq u_1\), since \(u_2\) is a least upper bound. Hence, \(u_1=u_2\).
  2. By contradiction. Suppose not. Then there exists \(\varepsilon > 0\) such that for all \(x \in X, \sup X - \varepsilon \geq x\). Then \(\sup X - \varepsilon\) is an upper bound with \(\sup X - \varepsilon < \sup X\) contradicting \(\sup X\) being supremum.

3.0.5 Def “max/min”

Fix \(X \subseteq \mathbb{R}\) nonempty

  1. Call \(\bar{x}\) a max of \(X\) if \(\bar{x} \in X\) and \(\bar{x} \geq x\) for each \(x \in X\)
  2. Call \(\underline{x}\) a min of \(X\) if \(\underline{x} \in X\) and \(\underline{x} \leq x\) for each \(x \in X\)

Example: [0,1) doesn’t have a max, even though it’s bounded

3.0.6 Prop 2 “max=sup”

Fix \(X \subseteq \mathbb{R}\) nonempty

  1. \(\bar{x}\) is a max of \(X\) iff \(\bar{x}=\sup X \in X\)
  2. \(\underline{x}\) is a max of \(X\) iff \(\underline{x}=\inf X \in X\)

Proof: a) \(\bar{x} \in X\), we know this from the def of the max. b) need to show \(\bar{x}=\sup X\). Well, we know \(\bar{x}\) is an upper bound and there exists a least upper bound \(=sup X\) by the completeness axiom. Since \(\bar{x} \in X\) each upper bound of \(X\) is greater than or equal to \(\bar{x}\). Hence, \(\bar{x}=\sup X\).

3.0.7 Corollary 1:

Fix \(X \subseteq \mathbb{R}\) nonempty. Then \(X\) has a max iff \(\sup X \in X\). Moreover, if \(X\) has a max, it is unique.

A real-valued function, \(f: X \rightarrow \mathbb{R}\). Call \(f\) bounded if \(f(X) \subseteq \mathbb{R}\) is bounded.

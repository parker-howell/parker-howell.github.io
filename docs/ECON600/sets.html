1 Sets

1.0.0.1 Class 1: 2023-08-14

1.0.1 Def “Properties of a set”

  • order doesn’t matter
  • redundancies don’t matter
    • \(X=Y\) means (by def) \(x \in X\) iff \(x \in Y\)

1.0.2 Def: “Subsets”

\(X \subseteq Y\) if for each \(x\in X, x \in Y\).

1.0.3 Remark 1 “proper subset”

If \(X \subseteq Y\) and it is not the case that \(Y \subseteq X\) then we say \(X\) is a proper subset of \(Y\). We denote this as \(X \subsetneq Y\).

1.0.4 Remark 2 “empty set in all sets”

For any set \(X, \emptyset \subseteq X\)

1.0.5 Important Object “power set”

The set of all subsets of \(X = 2^X\)

Example if \(X = \{a, b, c\}\) then \(2^X = \{\emptyset, \{a\}, \{b\}, \{c\}, \{a,b\}, ..., \{a,b,c\}\}\)

1.0.6 Operations on Sets

  1. Union \(X \cup Y = \{x: x \in X \text{ or } x \in Y\}\) (or for an arbitrary number of sets: \(\cup_{i\in I} X_i=\{x:x\in X_i \text{ for some } i \in I\}\))
  2. Intersection \(U \cap Y = \{x: x \in X \text{ and } x \in Y\}\) (or for an arbitrary number of sets: \(\cap_{i\in I} X_i=\{x:x\in X_i \text{ for all } i \in I\}\))
  3. Difference: \(Y \setminus X = \{x \in Y: x \not\in X\}\)

\(E \subseteq X\): Complement of \(E\): \(E^C = X\setminus E\)

  • Say \(X\) and \(Y\) are disjoint if \(X \cap Y=\emptyset\).
  • Say \((X_i: i \in I))\) are disjoint if \(\cap X_{i\in I} = \emptyset\)
  • Say pairwise disjoint if for each \(i\neq j, X_i \cap X_j = \emptyset\).

Caution: \(X_1 = \{0,1,2\}, X_1 = \{2,3\}, X_1 = \{3,0\}\) are disjoint, but not pairwise disjoint.

1.0.7 Def “Partition”

\(P \subseteq 2^x\) is a partition of \(X\) if the following hold:

  1. if \(E, F \in P\) and \(E\cap F\neq \emptyset\) then \(E=F\)
  2. \(U_{E_u\in P}E_i=X\)

1.0.8 Def “Cartesian Product”

For \(X_i: i\in I\):

\(\Pi_{i \in I} X_i = \{(x_i: i\in I): x_i \in X_i\}\)

Caution: never use curly braces when working with a Cartesian product. Order matters in a Cartesian product.

If \((X_i: i = 1, ... ,n)\) where \(X_i=X\) for each \(i=1, ..., n\), then \(X_n=\Pi_{i=1}^n X_i\). We call this the n-fold product.

1.1 Relations

Relation on \(X \times Y: R\) : idea is that the relation, \(R\), expresses a relationship between members of \(X\) and members of \(Y\).

Example:

  • \(\geq\) is a relation on \(\mathbb{R} \times \mathbb{R}\) such that \(x\geq y\) iff \(x\) is greater than or equal to \(y\)
  • \(>\) is a relation on \(\mathbb{R} \times \mathbb{R}\) such that \(x > y\) iff \(x\) is strictly greater than \(y\)

We can view \(\geq\) as a subset of \(\mathbb{R} \times \mathbb{R}\). In notation: \(\{(x,y) \in \mathbb{R} \times \mathbb{R}: x \geq y\}\)

A relation, \(R\), on \(X \times Y\) is some \(R \subseteq X \times Y, (x,y) \in R\). Can also write \(xRy\).

Possible properties of a relation, \(R\), on \(X \times X\):

  1. Reflexive: for each \(x \in X, xRx\)
  2. Transitive: if \(xRy\) and \(yRz\) then \(xRz\).
  3. Symmetric: if \(xRy\) then \(yRx\)
  4. Antisymmetric: if \(xRy\) and \(yRx\) then \(x=y\).
  5. Complete: for each \(x, y \in X\), either \(xRy\) or \(yRx\).

Example: \(\geq\) is

  1. reflexive
  2. transitive
  3. NOT symmetric
  4. antisymmetric (only way for to be greater than y and y greater than x is for them to be equal)
  5. complete (yep, can order any two numbers)

Example: \(>\) is

  1. NOT reflexive
  2. transitive
  3. NOT symmetric
  4. YES antisymmetric (trivially - the sufficient condition is not satisfied (impossible for \(xRy\) and \(yRx\) to happen here), so the necessary condition is irrelevant)
  5. NOT complete (can’t order the same number)

Example: \(\geq\) on \(\mathbb{R} \times \mathbb{R}\), such that \((x_1,x_2) \geq (y_1,y_2) \iff x_1 \geq y_1, x_2 \geq y_2\). This is

  1. reflexive
  2. transitive
  3. NOT symmetric
  4. antisymmetric
  5. NOT complete (cannot rank ordered pairs)

1.1.1 Def “Equivalence Relation”

An Equivalence Relation on \(X \times X\) is 1. reflexive 2. symmetric 3. transitive

Example: \(X = \{a,c,l\}\). Preference relationship on \(X \times X: R\) is \(aRl, lRA, aRA, lRl, cRc, aRc, lRc\).

This is not an equivalence relation because it is not symmetric. But, I can derive an indifference relationship, \(I\) from \(R\): a\(I\)l, l\(I\)a, a\(I\)a, l\(I\)l, c\(I\)c. And this is an equivalence relation!
